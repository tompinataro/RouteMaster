Bloom Steward — Support

Need help? Start with the options below.

Contact

  • Email (preferred): TixPy@pinataro.com
  • GitHub Issues: Open a ticket

Quick Checks

  • Server Health: verify the API `/health` endpoint (see the Staging Runbook).
  • App Version: confirm you are on the latest TestFlight build.
  • Network: ensure a reliable connection before retrying submissions.

Common Topics

  • Login & Accounts: Organization-managed accounts only; contact your admin for access.
  • Submissions: Offline submissions queue locally and auto-sync when connectivity returns.
  • Location: Permission is optional; flows work without it if your org does not require proof-of-presence.
